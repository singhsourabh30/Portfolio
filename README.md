## üì¶ Installation

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Setup

```bash
# Clone repository
git clone <repository-url>
cd Portfolio

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm build

# Start production server
npm start
```

The application will be available at `http://localhost:3000`

## üé® Customization

### Colors
Edit `tailwind.config.ts` to modify neon colors:
```typescript
colors: {
  neon: {
    blue: '#00D9FF',
    cyan: '#00FFFF',
    purple: '#A020F0',
    green: '#39FF14',
  }
}
```

### Content
Update component files to modify:
- **Hero Section**: `src/components/HeroSection.tsx`
- **Terminal Commands**: `src/components/TerminalSection.tsx`
- **Experience**: `src/components/ExperienceSection.tsx`
- **Projects**: `src/components/ProjectsSection.tsx`
- **AI Responses**: `src/components/AIAssistant.tsx`

### Animations
Adjust Framer Motion settings in individual components for custom animation speeds and effects.
